Projeto desenvolvido com react, type script e tawilwinsd

O proposito desse projeto é apresentar um ecommerce funcional que atenda as demandas de um negocio com regras de adiocionar,
remover e apresnetar os resultados no carrinho

FUNCIONALIDADES E SUAS BIBLIOTECAS
1- redux para gerenciar os dados no carrinho

2- clerck para gerenciar rotas e acesso permitido

3- jsonServer como baco de dados

4- s3 da aws para arazenar imagens

5- react-con para pegar icones
